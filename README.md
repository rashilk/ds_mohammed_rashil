# Trader Behavior vs Market Sentiment Analysis (Phase 1)

This project analyzes cryptocurrency market sentiment using the Bitcoin Fear & Greed Index.

Due to the temporary unavailability of trader-level transaction data, this phase focuses on sentiment trends and inferred behavioral insights. The project is designed to integrate trader data once available.

## Project Structure
- notebook_1.ipynb – Google Colab notebook for analysis
- csv_files/ – Cleaned datasets
- outputs/ – Visualizations and charts
- ds_report.pdf – Final summarized report

This project was completed as part of the Data Science Internship assignment for Primetrade.ai.
